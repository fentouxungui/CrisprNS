# CrisprNS
Crispr Negative Screen
A simple analysis package for Crispr Negative Screen using MAGeCK output as input!

## Install

```r
if(!require("CrisprNS")){
  library("devtools")
  devtools::install_github("fentouxungui/CrisprNS")
}
library(CrisprNS)
```
